Project Overview

This is a professional, fully responsive website for a fictional environmental NGO called Bright-World. 
The website was developed using HTML5 and CSS only, without any JavaScript frameworks or CSS preprocessors, meeting all the technical requirements of the midsemester project.


Features Implemented

HTML5 Semantic Elements
- `<header>` - Navigation and branding
- `<nav>` - Navigation menu
- `<main>` - Primary content
- `<section>` - Content sections
- `<article>` - Blog-style content
- `<aside>` - Sidebar content (footer sections)
- `<footer>` - Footer with contact and links
- `<table>` - Data presentation
- `<form>` - Contact and volunteer forms
- `<img>` - Image elements
- `<a>` - Hyperlinks
- `<ul>`, `<li>` - Lists

 CSS Features
- Flexbox Layout - Navigation bars, card layouts, button groups
- CSS Grid - Project cards, gallery grid, stats section
- Responsive Design - Media queries for mobile, tablet, and desktop
- Hover Effects - Interactive buttons and cards
- Transitions & Animations - Smooth hover states and transformations
- Card Components - Styled project and testimonial cards
- Shadows & Rounded Corners - Modern visual styling
- External Stylesheet - All styles in `css/style.css`
- CSS Variables - Color scheme and spacing management

 Responsive Breakpoints
- Desktop: 1200px and above
- Tablet: 768px - 991px
- Mobile: Below 768px
- Small Mobile: Below 480px

 Pages Description

 1. Home Page (index.html)
- Hero section with call-to-action buttons
- Mission and vision cards
- Featured projects showcase
- Impact statistics
- Testimonials section
- Partners section

2. About Us Page (about.html)
- Organization history
- Vision and mission statements
- Core values
- Strategic objectives (table format)
- Leadership team
- Why the organization exists

3. Projects Page (projects.html)
- Six featured projects with:
  - Project images
  - Title and description
  - Location information
  - Current status
  - Impact metrics

 4. Gallery Page (gallery.html)
- CSS Grid-based image gallery
- 8 high-quality images
- Hover overlay effects
- Responsive grid layout

5. Volunteer Page (volunteer.html)
- Volunteer opportunities list
- Registration form with fields:
  - Full Name
  - Email
  - Phone
  - Gender (dropdown)
  - Age
  - Skills (textarea)
  - Availability (dropdown)

 6. Donate Page (donate.html)
- Why support section
- Donation impact statistics
- Donation form with:
  - Amount selection
  - Donor information
  - Payment method selection
  - Optional message field

 7. Contact Page (contact.html)
- Contact information (address, phone, email, hours)
- Social media links
- Contact form with:
  - Name, email, subject, message
  - Form validation
- Map placeholder
